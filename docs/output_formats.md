# Output File Formats

This document provides detailed documentation for all TSV output files generated by `final_finalizer`.

## Overview

| File | Description |
|------|-------------|
| `*.contig_summary.tsv` | Per-contig classification with metrics |
| `*.evidence_summary.tsv` | Synteny evidence per contig Ã— reference chromosome |
| `*.segments.tsv` | Individual synteny segments (protein alignments) |
| `*.macro_blocks.tsv` | Aggregated synteny chains/macro-blocks |
| `*.ref_lengths.tsv` | Reference chromosome lengths |

---

## contig_summary.tsv

The main output file containing classification results and quality metrics for each query contig.

### Classification Columns

| Column | Type | Description |
|--------|------|-------------|
| `contig` | string | New contig name (reflects chromosome assignment, e.g., `chr5A` or `contig_1`) |
| `original_name` | string | Original contig name from input FASTA |
| `classification` | string | Classification category (see [Categories](#classification-categories)) |
| `classification_confidence` | string | Confidence level: `high`, `medium`, or `low` (see [Confidence](#classification-confidence)) |
| `reversed` | yes/no | Whether the contig was reverse-complemented to match reference orientation |
| `contaminant_taxid` | integer | NCBI taxonomy ID (only for `contaminant` classification) |
| `contaminant_sci` | string | Scientific name of contaminant organism |

### Evidence Strength Columns

These columns provide the underlying evidence used to determine classification confidence.

| Column | Type | Description |
|--------|------|-------------|
| `gc_content` | float | GC content of contig (0.0-1.0) |
| `gc_deviation` | float | How many standard deviations the contig GC differs from reference mean |
| `synteny_score` | float | Synteny evidence strength (0.0-1.0); based on gene proportion |
| `contam_score` | float | Contaminant evidence strength (0.0-1.0); normalized centrifuger score |
| `contam_coverage` | float | Fraction of contig covered by contaminant alignments (0.0-1.0) |

### Assignment Columns

| Column | Type | Description |
|--------|------|-------------|
| `assigned_subgenome` | string | Subgenome identifier (e.g., `A`, `B`, `At`, `Dt`) or `NA` if none |
| `assigned_ref_id` | string | Best-matching reference chromosome ID (e.g., `chr5A`) or `NA` |
| `assigned_chrom_id` | string | Chromosome number without subgenome (e.g., `chr5`) |
| `status` | string | Assignment status: `OK`, `NO_HITS`, `AMBIG_LOW_FRAC`, or `AMBIG_LOW_RATIO` |

### Size and Coverage Columns

| Column | Type | Description |
|--------|------|-------------|
| `length` | integer | Contig length in base pairs |
| `best_ref_union_bp` | integer | Total aligned base pairs to best reference (union of all alignments) |
| `best_ref_union_frac` | float | Fraction of contig covered by alignments to best reference (`best_ref_union_bp / length`) |
| `total_aligned_bp` | integer | Total aligned base pairs across all reference chromosomes |
| `n_ref_hits` | integer | Number of distinct reference chromosomes with alignments to this contig |

### Scoring Columns

| Column | Type | Description |
|--------|------|-------------|
| `best_score` | float | Synteny score for best reference assignment |
| `second_score` | float | Synteny score for second-best reference |
| `score_ratio` | float | Ratio of best to second score (`best_score / second_score`); `Inf` if no second hit |

### Gene Statistics Columns

| Column | Type | Description |
|--------|------|-------------|
| `ref_gene_proportion` | float | Proportion of reference chromosome genes aligned to this contig |
| `genes_per_Mbp` | float | Gene density: aligned reference genes per megabase of query sequence |

### Quality Flag Columns

| Column | Type | Description |
|--------|------|-------------|
| `chimeric` | yes/no | Whether the contig appears to be chimeric (significant alignments to multiple chromosomes) |
| `chimera_reason` | string | Details of chimera detection (e.g., `primary_frac=0.650,secondary_frac=0.250`) |
| `low_coverage` | yes/no | Whether total aligned bp is below `--chr-like-minlen` threshold |

### Best Reference Alignment Metrics

| Column | Type | Description |
|--------|------|-------------|
| `best_matches` | integer | Total matching bases in alignments to best reference |
| `best_aln_len` | integer | Total alignment length to best reference |
| `best_identity` | float | Alignment identity to best reference (`best_matches / best_aln_len`) |
| `best_distance` | float | Alignment distance to best reference (`1 - best_identity`) |

### Second-Best Reference Alignment Metrics

| Column | Type | Description |
|--------|------|-------------|
| `second_ref_id` | string | Second-best reference chromosome ID or `NA` |
| `second_ref_union_bp` | integer | Total aligned base pairs to second-best reference |
| `second_matches` | integer | Total matching bases in alignments to second-best reference |
| `second_aln_len` | integer | Total alignment length to second-best reference |
| `second_identity` | float | Alignment identity to second-best reference |
| `second_distance` | float | Alignment distance to second-best reference |

### Classification Categories

| Category | Description |
|----------|-------------|
| `chrom_assigned` | Chromosome-length contig assigned to a reference chromosome |
| `chrom_unassigned` | Chromosome-length contig without reference assignment (novel or failed synteny gates) |
| `organelle_complete` | Complete organelle genome (chrC or chrM) |
| `organelle_debris` | Partial organelle sequence |
| `rDNA` | Ribosomal DNA repeat unit |
| `contaminant` | Sequence from contaminating organism |
| `chrom_debris` | High-coverage (â‰¥80%), high-identity (â‰¥90%) duplicate of an assembled chromosome |
| `debris` | Assembly debris with reference homology (â‰¥50% coverage) or protein hits (â‰¥2) |
| `unclassified` | Could not be classified (below chromosome-length threshold) |

### Classification Confidence

The `classification_confidence` column indicates how certain the classification is, based on multiple lines of evidence.

| Level | Criteria |
|-------|----------|
| `high` | Strong, consistent evidence supporting the classification |
| `medium` | Moderate evidence, or some conflicting signals |
| `low` | Weak evidence, conflicting signals, or unusual characteristics |

**Evidence factors considered:**

- **Synteny strength**: Gene proportion and number of synteny segments
- **GC content**: Deviation from reference nuclear chromosomes (in standard deviations)
- **Contaminant coverage**: Fraction of contig covered by contaminant alignments

**Per-category confidence logic:**

| Classification | High | Medium | Low |
|---------------|------|--------|-----|
| `chrom_assigned` | Gene proportion â‰¥20%, GC deviation <2Ïƒ | Gene proportion 10-20%, or GC deviation 2-3Ïƒ | Gene proportion <10%, or GC deviation >3Ïƒ |
| `chrom_unassigned` | â€” | GC deviation <2Ïƒ | GC deviation â‰¥2Ïƒ |
| `contaminant` | Coverage â‰¥80%, or GC deviation >2Ïƒ | Coverage 50-80% | Coverage <50% |

### Assignment Status Values

| Status | Description |
|--------|-------------|
| `OK` | Contig passed all assignment criteria |
| `NO_HITS` | No synteny evidence found |
| `AMBIG_LOW_FRAC` | Best reference coverage below `--assign-min-frac` threshold |
| `AMBIG_LOW_RATIO` | Score ratio below `--assign-min-ratio` threshold |

## evidence_summary.tsv

Synteny evidence aggregated per contig Ã— reference chromosome pair. Use this to understand the strength of evidence supporting each potential chromosome assignment.

| Column | Type | Description |
|--------|------|-------------|
| `contig` | string | Query contig name |
| `ref_id` | string | Reference chromosome ID |
| `chrom_id` | string | Chromosome number without subgenome |
| `subgenome` | string | Subgenome identifier or `NA` |
| `union_bp` | integer | Union of aligned base pairs (non-overlapping) |
| `matches` | integer | Total matching bases across all chains |
| `aln_len` | integer | Total alignment length across all chains |
| `identity` | float | Overall alignment identity (`matches / aln_len`) |
| `gene_count` | integer | Number of unique reference genes with alignments |
| `n_chains_kept` | integer | Number of synteny chains retained after filtering |
| `score_topk` | float | Sum of top-K chain scores (controlled by `--assign-chain-topk`) |
| `score_all` | float | Sum of all chain scores |
| `best_chain_weight` | float | Score of the best individual chain |
| `best_chain_qbp` | integer | Query base pairs in best chain |
| `best_chain_identity` | float | Identity of best chain |

## segments.tsv

Individual synteny segments (protein alignment anchors). Each row represents one protein alignment that passed quality filters.

| Column | Type | Description |
|--------|------|-------------|
| `contig` | string | Query contig name |
| `contig_len` | integer | Query contig length |
| `target_chrom_id` | string | Reference chromosome number |
| `target_subgenome` | string | Subgenome identifier or `NA` |
| `strand` | +/- | Alignment strand |
| `chain_id` | integer | ID of the chain this segment belongs to |
| `qstart` | integer | Query start position (0-based) |
| `qend` | integer | Query end position (0-based, exclusive) |

---

## macro_blocks.tsv

Aggregated synteny chains (macro-blocks). Each row represents one collinear chain of protein alignments.

| Column | Type | Description |
|--------|------|-------------|
| `contig` | string | Query contig name |
| `contig_len` | integer | Query contig length |
| `ref_id` | string | Reference chromosome ID |
| `chrom_id` | string | Chromosome number without subgenome |
| `subgenome` | string | Subgenome identifier or `NA` |
| `strand` | +/- | Predominant strand of the chain |
| `chain_id` | integer | Unique chain identifier |
| `qstart` | integer | Query start position of chain |
| `qend` | integer | Query end position of chain |
| `qspan_bp` | integer | Query span (`qend - qstart`) |
| `union_bp` | integer | Union of aligned bases within chain |
| `matches` | integer | Total matching bases in chain |
| `aln_len` | integer | Total alignment length in chain |
| `identity` | float | Chain alignment identity |
| `score` | float | Chain score (used for ranking) |
| `n_segments` | integer | Number of segments in this chain |
| `gene_count_chain` | integer | Number of unique genes in this chain |

## Usage Tips

### Filtering for High-Confidence Assignments

```bash
# Get chromosome contigs with status OK
awk -F'\t' 'NR==1 || ($3=="chrom" && $10=="OK")' output.contig_summary.tsv

# Get contigs with high gene density (>50 genes/Mbp)
awk -F'\t' 'NR==1 || ($18 > 50)' output.contig_summary.tsv
```

### Identifying Problematic Contigs

```bash
# Find chimeric contigs
awk -F'\t' 'NR==1 || $21=="yes"' output.contig_summary.tsv

# Find ambiguous assignments
awk -F'\t' 'NR==1 || $10 ~ /AMBIG/' output.contig_summary.tsv
```

### Analyzing Synteny Evidence

```bash
# Get all evidence for a specific contig
awk -F'\t' '$1=="contig_name"' output.evidence_summary.tsv

# Find contigs with evidence to multiple subgenomes
awk -F'\t' 'NR>1 {print $1}' output.evidence_summary.tsv | sort | uniq -c | awk '$1>1'
```
