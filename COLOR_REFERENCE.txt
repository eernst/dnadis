================================================================================
COLOR BLINDNESS COMPATIBILITY
================================================================================

The Okabe-Ito palette is designed for maximum distinguishability in:
  ✓ Protanopia (red-blind)
  ✓ Deuteranopia (green-blind)
  ✓ Tritanopia (blue-blind)
  ✓ Grayscale printing

Test your plots at: https://www.color-blindness.com/coblis-color-blindness-simulator/

================================================================================
IMPLEMENTATION NOTES
================================================================================

1. All templates now use ggokabeito::palette_okabe_ito()
2. Light variants generated with colorspace::lighten(color, amount = 0.5)
3. Subgenome positions 7-8 reserved for high-ploidy genomes
4. Classification palette uses all 9 Okabe-Ito colors
5. For >9 classifications, use lighten() to create variants
6. For >8 subgenomes, templates will warn and truncate to first 8
